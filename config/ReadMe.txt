	Добро пожаловать в проект реализация клиент - серверного приложения с использованием языка программирования Python и фреймворка для веб-разработок Django, предназначенного для сбора,управления и упорядочивания различных данных, используемых на различных веб-ресурсах.
Предусмотреть реализацию RESTFull API для интеграции проекта в другие сервисы без нарушения целостности данных.

	=== Системные требования ===
Операционная система : Linux
Жесткий диск : 40 МБ свободного места

	=== Развертывание проекта ===
1) На удаленном VDS сервере необходимо установить все компоненты прописанные в файле requirements.txt
2) Создать базу данных для хранения данных проекта (python manage.py syncdb; python manage.py makemigrations; python manage.py migrate)
3) Запустить сервер (python manage.py runserver)

	=== Инструкция пользователя ===
1) На главной странице проекта создать личный кабинет, нажав на кнопку "Создать аккаунт"
2) Если аккаунт существует, авторизироваться
3) Получаем доступ к человеко-ориентированному интерфейсу
4) Выбираем доступные пункты меню для отправки сообщений или редактирование базы пользователей
